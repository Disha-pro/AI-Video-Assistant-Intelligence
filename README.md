# 🎬 AI Video Assistant Intelligence

An AI-powered application that transforms meeting recordings and YouTube videos into actionable insights using Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and Speech-to-Text technologies.

---
## ✨ LIVE STREAMLIT APP LINK 


## ✨ Features

- 🎥 Analyze YouTube videos or local meeting recordings
- 🎙 Speech-to-Text using Whisper
- 🌐 English & Hinglish support
- 📝 Automatic Meeting Summaries
- 📌 AI Generated Meeting Title
- ✅ Action Item Extraction
- 🔑 Key Decision Detection
- ❓ Open Question Extraction
- 💬 Chat with Meeting using RAG
- ⚡ Beautiful Streamlit Dashboard

## Installation

```bash
git clone https://github.com/USERNAME/AI-Video-Meeting-Intelligence.git

cd AI-Video-Meeting-Intelligence

pip install -r requirements.txt

streamlit run app.py
```

## Environment Variables

Create a `.env` file

```
MISTRAL_API_KEY=

SARVAM_API_KEY=

WHISPER_MODEL=base

SARVAM_STT_MODEL=saaras:2.5
```
